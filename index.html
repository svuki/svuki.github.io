The game proceeds state by state according to the following rules:

  • If a cell is alive with two or three neighbors, it remains alive. Otherwise it dies
  • If a cell is dead with three neighbors, it becomes alive. Otherwise it remains dead
A neighborhood is defined as the 8 cells immediately surrounding a given cell.

Controls

  • Click on a cell to toggle it on/off.
  • Drag across a region of cells to save the underlying pattern.
  • Shift click to copy the last saved pattern The pattern will then be oriented so that the upper left corner is at the click position.
  • The Go/Stop button will toggle iteration.
  • The slide controls the rate of iteration.
  • The reset button will clear the board of any live cells.
  • The grid button will overlay a grid over the board.
  • The save button will add the current state to the save-list. Clicking an element of the save-list will restore that state to the main canvas
